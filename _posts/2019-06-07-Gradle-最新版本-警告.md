### 背景
Android Studio 已更新至3.4.1，之前运行的好好的项目，Build后提示了很多警告。`... It will be removed at the end of 2019. ...`
如果你发现你的代码里没有相关的引用，很可能是三方库/插件代码有问题

本文旨在定位警告来源，解决办法还需跟进相关issue

#### 警告 variantOutput.getProcessResources

`ButterKnife` [github-issue](https://github.com/JakeWharton/butterknife/issues/1431)<br/>
WARNING: API 'variantOutput.getProcessResources()' is obsolete and has been replaced with 'variantOutput.getProcessResourcesProvider()'.
It will be removed at the end of 2019.
For more information, see https://d.android.com/r/tools/task-configuration-avoidance.
To determine what is calling variantOutput.getProcessResources(), use -Pandroid.debug.obsoleteApi=true on the command line to display a stack trace.
Affected Modules: account
